# QA-Project-Urban-Scooter.--API

# Proyecto # 9 Manejo de SQL, Postman y Android studio

## Descripción General

En este proyecto es la visualización desde back-end cara a un repartidor y un cliente en el servicio de renta de un scooter, por lo que acá se ve desde la creación de un usuario, la creación del pedido de un scooter y como obtener el número de tracking el aceptar el pedido o eliminar el mismo
